Palindrome Project
Objective

The objective of this project is to check whether a given word or number is a palindrome (reads the same forward and backward).

Project Flow

User enters a word or number.

The system reverses the input.

The system compares the original and reversed values.

If both are the same, it displays “Palindrome”; otherwise, “Not a Palindrome”.

Use Case

This project is useful for understanding string handling, logic building, and basic programming concepts.
